Plunk Home Valuation Widget

How it works?

This widget is intended to be used as a complement to your pages, use it in a sidebar or in a section of your site. The widget has a search bar that uses Google's auto-complete service to help users place the correct address.

The first step is for the visitor to type the address he/she wants to know the valuation, and choose one of the results provided by google, then click on the submit button.

A modal window will open and the address search will be performed on more than 104 million homes nationwide.

There can be two results, a correct match was found and the value of the property will be displayed in real time, or there can be the case where no match is found.

Correct match

In the following image you can see the content of a result, the home ticker at the top and a card at the bottom.

This card asks for a phone number in order to send market insights for that zip code.

No results found

It may be the case that you do not find any results that match the address provided, when that happens you will see a window like this one, in the same way a phone number is requested so that an agent can contact the lead and give him/her a personalized attention.:

When the user enters his phone number and presses submit the agent will receive a text message with the lead's phone number and the requested address.

Market Insights

When a customer requests market insights, a link is sent to a landing page where they can view the market insights of the requested zipcode. It looks like the following

You can modify the content of the landing page in the advanced settings of the widget, you can add an image preferably a logo, you can also modify the header and footer of the page.

You can see the list of requests at the bottom of the widget configuration page.
