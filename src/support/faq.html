How do I add IDXAddons widgets on a Wix website?

To add an IDXAddons widgets to a Wix website, follow these steps:

  1. Log in to your Wix account and navigate to the page where you want to add the widget.
  2. Click on the "Add" button on the left-hand side of the screen and select "More" from the menu.
  3. In the search bar, type "HTML iframe" and select the "HTML iFrame/Embed" option.
  4. Drag the "HTML iFrame/Embed" element to the desired location on the page.
  5. Click on the "Enter Code" button in the element's settings, and paste the IDXAddons widget code inside a paragraph HTML tag. For example, <p>WIDGET_CODE_HERE</p>.
  6. Adjust the widget size and other settings as needed, and then save and publish your website.

Note that you will need to have an active IDXAddons account and have generated the widget code from your account dashboard.

What is the IDXAddons Webhook and what is it used for?

A Webhook is a piece of code you'll need to set up to get the Text Alerts feature and the daily email report to function properly. It's like a messenger between your website and IDXAddons that sends information back and forth.

To set it up, log in to your IDX Broker account and turn off the WYSIWYG button. Then, copy the code from the Webhook page at IDXAddons.com and paste it into the IDX Broker Sub-header page.

Once you've saved the page, you can check that the Webhook is working properly on the IDXAddons Webhook page using the "Validate" button.
